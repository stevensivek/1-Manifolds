# Formalization of the Topological Classification of 1-Manifolds

This project formalizes the topological classification of connected 1-manifolds in the [Lean theorem prover](https://lean-lang.org/).  The main theorem is that any connected, second countable, Hausdorff space equipped with an atlas of charts to $\mathbb{R}^1$ is homeomorphic to either $\mathbb{R}^1$ or a circle.

The proof is based on the argument in Chapter 3, Section 1, Theorem 17 of Fuks and Rokhlin's book "Beginner's Course in Topology", and builds on
Jeff Lee's [MSc project repository](https://github.com/jeffimp/Classification-of-Curves).
